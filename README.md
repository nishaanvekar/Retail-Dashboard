# Retail Business Dashboard

## Objective
Analyze sales performance across regions and categories.

## Tools Used
- Excel (Pivot Tables, Charts, Slicers)

## Insights
- West region generates highest revenue
- Electronics category most profitable
- High discounts reduce profit
